# App（重写，一开始就进入登录注册）
### 注册
1. 在 dialog 内判断输入内容
2. 按下注册键执行方法，先利用 `validate` 和 `prop` 判断给定的数据是否填入
3. true的情况下调用注册API

### 登录
1. 把账号密码发送到服务器
2. 收到返回结果,设置cookie、关闭dialog、信息栏改变
3. 登录的时候先判断发送cookie
4. `oncreate`的时候先判断有没有`token`，有则发送请求
